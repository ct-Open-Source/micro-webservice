# How to build

## Prerequisites

### Ubuntu

```
apt install libboost-thread-dev libboost-chrono-dev libboost-date-time-dev
```

### macOS 13 Ventura

```
brew install libboost-thread-dev libboost-chrono-dev libboost-date-time-dev
```

## Build

```
CMAKE_BUILD_TYPE=Release cmake .
cmake --build .
````
